Welcome to VIMMO's documentation!
VIMMO is a mini Python project for the department to integrate with NGTD and panelapp to provide information about genes linked to a certain Rcode. It pulls data from the Open Food Facts database and offers a simple and intuitive API.

Check out the :doc:`usage` section for further information, including how to :ref:`installation` the project.

Note

This project is under active development.
